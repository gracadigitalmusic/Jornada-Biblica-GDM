# Guia do Modo Co-op Online

O modo Co-op Online permite que você e seus amigos joguem juntos, respondendo às mesmas perguntas e compartilhando vidas e power-ups.

## Como Jogar em Equipe

Para iniciar uma sessão cooperativa, um jogador deve ser o **Host** (Anfitrião) e os outros devem **Entrar** na sessão.

### 1. Criar uma Sessão (Host)

O Host é responsável por criar a sala e iniciar o jogo.

1.  **Acesse o Menu Principal** e clique em **"Co-op Online"**.
2.  Você será levado ao **Lobby**. O sistema criará automaticamente uma sessão para você.
3.  **Anote o Código da Sessão** (um código de 8 caracteres, ex: `A1B2C3D4`) exibido no topo do Lobby.
4.  **Compartilhe este código** com seus amigos (via WhatsApp, Telegram, etc.).
5.  Aguarde seus amigos entrarem. Você verá os avatares e nomes deles aparecerem na lista de "Membros do Time".
6.  Quando todos estiverem no Lobby, cada jogador deve clicar em **"Estou Pronto"**.
7.  Quando todos os jogadores estiverem prontos, o Host deve clicar em **"Iniciar Jogo"**.

### 2. Entrar em uma Sessão (Convidado)

Os convidados precisam do código da sessão fornecido pelo Host.

1.  **Acesse o Menu Principal** e clique em **"Co-op Online"**.
2.  Na tela de configuração, você terá a opção de **"Entrar em uma Sessão"** (se esta opção ainda não estiver visível, o Host deve fornecer o código).
3.  **Insira o Código da Sessão** fornecido pelo Host.
4.  Você será adicionado ao Lobby.
5.  Clique em **"Estou Pronto"** para sinalizar ao Host que você está pronto para começar.
6.  Aguarde o Host iniciar a partida.

### 3. Comunicação e Gameplay

-   **Chat em Tempo Real:** Use o painel de **"Chat do Time"** no Lobby para conversar com seus companheiros antes do jogo.
-   **Vidas Compartilhadas:** O time compartilha um número fixo de vidas. Se alguém errar, a vida é deduzida do total do time.
-   **Turnos:** As perguntas são respondidas em turnos. Apenas o jogador cujo turno está ativo pode selecionar a resposta.
-   **Power-ups:** Os power-ups são compartilhados e devem ser usados estrategicamente pelo jogador no turno.
-   **Objetivo:** Trabalhem juntos para responder corretamente o máximo de perguntas possível e completar o capítulo (se estiverem jogando o Modo História Co-op).
